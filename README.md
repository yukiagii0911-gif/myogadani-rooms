# 茗荷谷キャンパス 教室ビューワー

加山勇気の個人開発プロジェクト。中央大学茗荷谷キャンパスの教室空き状況を
シラバスデータから可視化するWebアプリ。

## 何ができるか

- **教室の空き状況閲覧**: 法学部の全授業（2,290件）の時間割と教室を集約し、いつどの教室が空いてるかを可視化
- **マップ表示**: 建物の平面図上に教室を配置。色で授業中/空きを表示
- **フォロー機能（モック）**: 友達がどの教室にいるか・授業を取っているか表示（実装未完）

## 起動方法（Mac）

1. **`start.command` をダブルクリック**（ターミナルが開いてサーバ起動）
2. ブラウザで http://localhost:8765/ を開く
3. キャリブレーションツールは http://localhost:8765/calibrate/

止めるときはターミナルで `Ctrl+C` → 閉じる

## 起動方法（手動）

```bash
cd web
python3 -m http.server 8765
```

ブラウザで http://localhost:8765/

## スマホ実機テスト

同じWiFi内のスマホからアクセスする場合:

1. Macのローカルipを調べる: `ipconfig getifaddr en0`（例: 192.168.2.163）
2. スマホのSafariで `http://192.168.2.163:8765/`

## 教室位置のキャリブが難しい理由

1. 建物が不規則な多角形で、教室が斜め壁に沿って傾いてる
2. 公式の floorplan は加山さんが画像で持ってる（screenshots in conversation）
3. Excalidrawでトレース → 座標を floors.js に手打ち、という流れだったが効率悪い
4. キャリブレーションツール (`web/calibrate/`) を作って、画面で教室を配置 → JSONエクスポート の流れで進めることにした

**友達への依頼**: キャリブレーションツール (`http://localhost:8765/calibrate/`) を開いて、各階の教室を建物の中に配置していってほしい。
配置できたら「JSONコピー」ボタンを押して、加山さんに送ってもらえれば反映します。

### 既に配置済みの階

- **B1**: 4教室 (B1C16, B1E04, B1W01, B1W02)
- **1F**: 1教室 (1W01 = 特大教室)

初めてキャリブを開くと、上記が自動で読み込まれて表示されます (`web/calibrate/seed.json`)。
これらは「お手本」として置いてあるので、必要なら微調整してOK。

### 残りやってほしい階

- **2F**: 10室 (2C02, 2C03, 2E02-2E08, 2W01+2W02=2BIG)
- **3F**: 16室 (3N02, 3N04, 3C02, 3C03, 3C13, 3E02-3E10, 3W01+3W02=3BIG)
- **5F**: 24室 (5N01-06, 5C02-5C11(5C05/07なし), 5W01-10)
- **6F**: 25室 (6C03-16(06なし), 6W01-12)

優先度: **5F が最重要**（自習でよく使う階・教室多い）→ 3F → 2F → 6F の順がオススメ。

## キャリブレーションツールの使い方

1. 階タブで対象階を選ぶ（B1〜6F）
2. 下の教室リストから1つタップ
3. 建物の中の正しい場所をタップ → 矩形が配置される
4. ドラッグで移動、スライダーで幅・高さ・角度を調整
5. 角度のクイックボタン（↺ 5° / ↺ 15° など）も使える
6. 「次の教室 →」ボタンで未配置の教室に進む
7. 全部終わったら右上「JSONコピー」→ コピーした内容を加山さんに送る

データはブラウザの localStorage に自動保存されるので、途中で閉じても続きから再開できる。
